# KVector

- Custom implementation of the **std::vector**.

## Building
- Visual Studio 17 2022:
  ```shell
  git clone https://github.com/mnink275/KVector
  cd KVector
  cmake -S . -B build
  cmake --build build
  cmake --open build
  ```

- Linux:
  ```shell
  git clone https://github.com/mnink275/KVector
  cd KVector/
  make build-debug
  ./build/KVector
  ```
